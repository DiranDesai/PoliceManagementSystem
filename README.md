
Police Management System
A web-based application designed to streamline the management of police operations, including handling cases, managing criminal records, and maintaining officer profiles.

🚀 Features
User Authentication: Secure login system for authorized personnel.

Case Management: Add, view, and update case details efficiently.

Criminal Records: Maintain comprehensive records of criminals.

Officer Profiles: Manage and update officer information.

Responsive Design: Accessible on various devices with a responsive layout.

💡 Usage
Login: Navigate to login.html to access the login page.

Dashboard: After logging in, you'll be directed to the main dashboard (index.html).

Manage Cases: Use cases.html to add or view case details.

Criminal Records: Access criminals.html to manage criminal information.

Officer Profiles: Visit profile.html to view or update officer details.
